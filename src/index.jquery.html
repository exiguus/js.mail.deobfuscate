Mail deobfuscate

jQuery Plugin

Mail deobfuscating with rot13 ceasar cipher

The plugin deobfuscate a mailto-link which was obfuscated with rot13 ceasar cipher onClick in this example

Example:

E-Mail with rot13 ceasar cipher ciphed:

  • with obfuscated mail address in href
  • znvy@rknzcyr.bet with obfuscated mail address in text
  • znvy@rknzcyr.bet with obfuscated mail address in text and title

Mailto Links


  <ul>
    <li>
      <a href="mailto:znvy@rknzcyr.bet" title="mail to">with obfuscated mail address in href</a>
    </li>
    <li>
      <a href="mailto:znvy@rknzcyr.bet" title="mail to">znvy@rknzcyr.bet with obfuscated mail address in text</a>
    </li>
    <li>
      <a href="mailto:znvy@rknzcyr.bet" title="mail to znvy@rknzcyr.bet">znvy@rknzcyr.bet with obfuscated mail address in text and title </a>
    </li>
  </ul>

mailDeobfuscate event binding


  <script>
    $('a[href^="mailto:"]').each(function () {
      $(this).on('click', function() {
        $(this).mailDeobfuscate()
      })
    })
  </script>
