Mail deobfuscate

ECMAScript 6 Class and Module

Mail deobfuscating with rot13 ceasar cipher

The module deobfuscate a mailto-link which was obfuscated with rot13 ceasar cipher onClick in this example

Example:

E-Mail with rot13 ceasar cipher ciphed:

  • with obfuscated mail address in href
  • znvy@rknzcyr.bet with obfuscated mail address in text
  • znvy@rknzcyr.bet with obfuscated mail address in text and title

mailDeobfuscate event binding


  import mailDeobfuscate from './js/mail.deobfuscate.module'

  const elements = document.querySelectorAll('a')
  elements.forEach(function (element) {
    element.addEventListener('click', function () {
      mailDeobfuscate(this)
    })
  })
